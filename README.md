# Análisis de desempeño y robustez de los parámetros de una estrategia evolutiva para el problema de llenado de grafos

Este repositorio contiene el código y la documentación de mi tesis de grado, enfocada en el Problema del Relleno Mínimo (Minimum Fill-in Problem).
Enfoque del Proyecto
El Problema del Relleno Mínimo (MFP) es un desafío conocido en la teoría de grafos y la optimización combinatoria que busca determinar el conjunto mínimo de aristas que triangulan un grafo. Este problema es de gran importancia práctica, ya que surge en la solución numérica de sistemas de ecuaciones lineales simétricas y definidas positivas dispersas.
Debido a que calcular el relleno mínimo es un problema NP-completo, se justifica el uso de métodos heurísticos avanzados.
## Objetivos del Estudio
El proyecto aborda el MFP mediante la aplicación de un Algoritmo Evolutivo (AE) —un método de generar y probar— para la optimización.
El foco principal de la investigación reside en:
1. Investigación de Parámetros: Estudiar el impacto y la configuración de los parámetros del Algoritmo Evolutivo (también llamados parámetros de estrategia) para optimizar el proceso de búsqueda.
2. Análisis de Rendimiento: Evaluar rigurosamente el desempeño (efectividad y eficiencia) y la robustez de la solución encontrada, lo cual implica asegurar un rendimiento estable a pesar de la naturaleza estocástica del algoritmo.

El repositorio rastrea la evolución del código y la evidencia experimental obtenida durante este proceso de optimización heurística y análisis de resultados.
